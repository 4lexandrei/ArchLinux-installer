# ArchLinux-installer
Another ArchLinux install script

Warning!
Currently doesn't support:
- Nvidia GPU drivers.
- BIOS 

Script breakdown:
1) Preset partition
2) Install packages
3) GRUB Bootloader setup
4) System configuration

Planning to add:

- Support to zram
